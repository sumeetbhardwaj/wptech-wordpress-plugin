Page Redirection, Post Redirection, Other Page Url Redirection

== Description ==

Page Redirection, Pages, Post and Custom Posts easily redirect to other url. (draft, private, public, pending).


== Installation ==

1. Upload the `wptech_redirection-page-hit-counter` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.
